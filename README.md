# Workflow n8n — Formulario a Excel con IA

## Descripción
Workflow automatizado desarrollado en n8n que permite capturar datos 
enviados desde un formulario web (Nombre y Email), procesarlos a través 
de un agente de inteligencia artificial basado en OpenAI, y almacenarlos 
de forma automática en una tabla de Microsoft Excel 365 alojada en OneDrive.
